\doxysubsection{Detailed Description}
This file sets up the connection to the database for Borrowing Function. 

It defines database credentials (servername,username,password and database name) then perform to establish a connection with the database.\+It checks whether the connection is successful or not if not, it terminates the script and displays the error message.
